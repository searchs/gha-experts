# README for DevOps Highland

This repository contains the code for the DevOps Highland project. The project is a simple API that returns the current time in UTC. The API is built with FastAPI and deployed on Railway. The frontend is built with React and Vite. The project is tested with Vitest and React Testing Library. The project is linted with Ruff and formatted with Prettier. The project is committed with Commitizen and versioned with uv. The project is deployed with Railway. The project is monitored with Sentry. The project is documented with MkDocs. The project is tested with GitHub Actions. The project is secured with Dependabot. The project is reviewed with CodeQL. The project is analyzed with SonarCloud. The project is built with GitHub Codespaces. The project is licensed under the MIT License.
